## instructions of use

run the command:

To Backup  
**node backup (name_of_collection)**

To Restore  
**node restore (name_of_collection)**

**_collections available:_**

- mal-user-token
- schedules
- users-discord
- users-info
- users-quiz

_note:_ to backup or restore all, pass "all".

_Example:<br/>_
**node backup all** or **node restore all**
